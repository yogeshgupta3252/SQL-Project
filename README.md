# Customer Purchase Project
Customer Purchase Portfolio Analysis using SQL

Project Overview
This project analyzes customer purchase data using SQL to identify sales trends, customer behavior, and revenue insights. The objective is to transform raw transactional data into meaningful business insights that can support decision-making.

Tools Used
MySQL


Key SQL Concepts Applied
Joins
Aggregate Functions (SUM, AVG, COUNT)
GROUP BY and HAVING
CASE Statements
Common Table Expressions (CTEs)
Window Functions
Subqueries


Business Questions Solved
Who are the top revenue-generating customers?
Which products contribute the highest sales?
What are the customer purchasing patterns?
Which categories generate the most revenue?
How can customers be segmented based on spending behavior?


Key Insights
Identified high-value customers contributing a significant portion of total revenue.
Analyzed product performance and category-wise sales trends.
Evaluated customer purchasing behavior using SQL-based analysis.
Generated actionable insights to support business reporting and decision-making.
